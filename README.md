# MySQL Cliente-Servidor en Ubuntu

Laboratorio técnico de instalación y configuración de MySQL en un entorno cliente-servidor con Ubuntu 24.04.

El proyecto documenta la instalación del servidor MySQL, la configuración para permitir conexiones remotas, la creación de usuarios y permisos, la conexión desde un cliente Linux y las pruebas con herramientas gráficas como MySQL Workbench y DBeaver.

## Objetivo

El objetivo de esta práctica es montar un entorno cliente-servidor funcional con MySQL, comprobando la conectividad remota, la autenticación de usuarios y el acceso a la base de datos desde diferentes clientes.

## Tecnologías utilizadas

* Ubuntu Server 24.04
* MySQL Server
* MySQL Client
* MySQL Workbench
* DBeaver
* SQL
* Git y GitHub
